# 📝 TODOLIST - Quản lý công việc cá nhân

Đây là một ứng dụng React (Create React App) giúp bạn quản lý công việc hằng ngày, gắn thẻ (tags), đánh dấu ưu tiên, tìm kiếm, và sắp xếp công việc.

---

## 🚀 Tính năng chính

- ✅ Tạo, chỉnh sửa, xoá task
- 🏷️ Gắn thẻ (tag) có màu cho task
- 🔍 Tìm kiếm theo nội dung, lọc theo thẻ hoặc ngày
- 📅 Lọc theo deadline
- 🔥 Sắp xếp theo độ ưu tiên hoặc hạn chót
- ☁️ Sử dụng MockAPI online (không cần backend cục bộ)

---

## 📦 Công nghệ sử dụng

- [React (Create React App)](https://create-react-app.dev/)
- Redux Toolkit & Redux Saga
- MockAPI: dùng làm giả server
- Docker + Nginx (production build)
